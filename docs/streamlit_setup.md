# DashIQ — Streamlit Setup

## Objective
To initialize a Streamlit application for dashboard visualization.

---

## Setup Steps

1. Created `app.py` inside `app/` directory
2. Installed required libraries:
   - streamlit
   - pandas
3. Connected dataset (CSV / SQLite)
4. Built initial layout:
   - Header
   - KPI metrics
   - Data preview

---

## Run Command

```bash
streamlit run app/app.py
